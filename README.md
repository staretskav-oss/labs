# Lab1 — Cubic Spline Interpolation

This project implements cubic spline interpolation for a mountain route profile.

## Files
- main.py — main program
- nodes.txt — GPS route data (distance and elevation)

## Functionality
The program:
- builds the route elevation profile
- constructs a cubic spline interpolation
- compares spline results for 10, 15 and 20 nodes
- calculates total ascent
- estimates mechanical energy required for the climb

## Output
The program generates:
- route elevation graph
- spline interpolation graph
- comparison of different node counts
- ascent statistics and energy calculation
